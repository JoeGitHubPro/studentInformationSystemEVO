
<h1>Student Management System</h1>

<h2>Description</h2>
<p>This program is a simple student management system that allows users to add students, courses, and student scores, as well as display student and course information.</p>

<h2>Features</h2>
<ul>
    <li>Add Student: Users can add new students with their details including student number, name, sex, age, institution, and major.</li>
    <li>Add Course: Users can add new courses with details such as course number, name, credit, and property (compulsory or optional).</li>
    <li>Add Student Score: Users can add scores for students in specific courses.</li>
    <li>Display Student Information: Users can view detailed information about a specific student by providing their student number.</li>
    <li>Display Course Information: Users can view detailed information about a specific course by providing its course number.</li>
    <li>Display Student Scores: Users can view scores of a specific student by providing their student number.</li>
</ul>

<h2>Usage</h2>
<ol>
    <li>Run the program.</li>
    <li>Choose the desired operation from the menu:
        <ul>
            <li>1: Add Student</li>
            <li>2: Add Course</li>
            <li>3: Add Student Score</li>
            <li>4: Display Student Information</li>
            <li>5: Display Course Information</li>
            <li>6: Display Student Scores</li>
            <li>7: Exit</li>
        </ul>
    </li>
    <li>Follow the prompts to perform the selected operation.</li>
    <li>After completing the operation, choose whether to perform another operation or exit the program.</li>
</ol>
